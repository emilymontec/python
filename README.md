## Python

Proyectos básicos para aprender y prácticar Python.

## Resumen
- **BasicAPI:** Idea de práctica para FastAPI, Uvicorn y Gunicorn
- **BasicWebApp:** Idea de práctica para Django
- **DecisionRoulette:** Juego de ruleta para toma de decisiones
- **GPyFile:** Gestor de archivos por medio de CLI (+GUI)
- **HMTL_API:** Web de práctica FastAPI + HTML
- **ReApp:** Web de práctica para comprender peticiones a las URLs
